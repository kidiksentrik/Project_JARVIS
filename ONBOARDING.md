# ⚡ Quick Onboarding Guide (New Device)

Follow these steps to bring JARVIS/FRIDAY online on a new machine.

## 1. Prerequisites
- Python 3.10+ installed.
- Git installed.

## 2. Bootstrapping
```bash
# Clone the repository
git clone https://github.com/kidiksentrik/Project_JARVIS.git
cd Project_JARVIS

# Run the interactive setup
python setup.py
```

## 3. Configuration Access
During setup, you will need keys from:
- **Gemini**: [Google AI Studio](https://aistudio.google.com/app/apikey)
- **Supabase**: [Supabase Project Settings](https://supabase.com/dashboard/project/_/settings/api)

## 4. Run Assistant
```bash
python main.py
```
